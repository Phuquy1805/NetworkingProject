## Problem 1 : Using TCP to download files


### Set up 
```bash
cd TCP
```

* Tạo thư mục ```server/server_files/``` và add file mà client muốn tải vào đó
* Di chuyển vào đường dẫn ```/server```


* Ghi các tên file client cần tải vào ```client/input.txt```
* Trên máy server
   ```bash
   cd server
   python server.py
   ```
* Trên máy client 
   ```bash
   cd client
   python client.py
   ```
